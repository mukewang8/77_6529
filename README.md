<h3>课程介绍：</h3>
<p>本课程是一门具有很强实践性质的“项目实战”课程，俗称“<a title="查看与 Java 相关的文章" target="_blank">Java</a>商城秒杀系统”，主要介绍了当前一些电商平台或者商城在举办一些商品秒杀、抢购活动时所涉及的相关业务流程，其中，主要包括“秒杀活动列表的展示”、“秒杀活动详情的获取”、“用户登录认证服务”、“高并发秒杀-抢购业务”、“秒杀成功订单的生成”以及“邮件通知服务”等等。</p>
